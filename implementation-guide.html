📋 Visão Geral

Verse Map é uma plataforma web completa para gestão musical de igrejas e organizações de louvor. A plataforma cobre desde a importação e análise automática de músicas até a execução ao vivo em cultos.

✨ Destaques

  • Importação de música por URL do YouTube com análise automática
  • Pipeline de análise assíncrona (BPM, tonalidade, estrutura)
  • Editor de arranjos versionado e imutável
  • Cifras editáveis com transposição de tom
  • Biblioteca de músicas por organização
  • Setlists com histórico imutável
  • Modo Live para apresentações
  • Multi-tenancy com isolamento completo
  • Dashboard operacional com métricas
  • Autenticação OAuth com controle de papéis

Pipeline de Análise

queued
  ↓
extracting_metadata (YouTube metadata)
  ↓
fetching_text (captions ou Whisper ASR)
  ↓
separating_sources (Demucs - opcional)
  ↓
analyzing_audio (Essentia: BPM, key, features)
  ↓
postprocessing_structure (LLM: labeling semântico)
  ↓
ready (pronto para edição)

🚀 Quick Start

1. Clonar Repositório

git clone https://github.com/luangitdev/verse-map-3.git
cd verse-map-3

2. Configurar Ambiente

cp .env.example .env
# Editar .env com suas variáveis

3. Iniciar Stack Completo

docker-compose up -d

4. Rodar Testes

pytest tests/ -v --cov=apps

5. Acessar Aplicação

  • Frontend: http://localhost:3000
  • API: http://localhost:8000
  • Flower: http://localhost:5555
  • pgAdmin: http://localhost:5050

⚠️ Requisitos

  • Docker & Docker Compose
  • Python 3.11+
  • Node.js 18+
  • Git
